### Drawing a dot
>

| Property      | Description                           |
| ------------  |-------------------------------------- |
| ctx           | 2D context from your canvas           |
| pos           | position of the object as a point     |
| scale         | scene scale                           |
| testing       | drawing point positions               |

```javascript
    draw: (ctx, obj, scale = 1) => {
            elementsTypes['layoutDot'].draw(ctx, objectUtils.point(100, 100), scale, true);
    }
```
