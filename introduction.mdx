## Setting up

This quick setup will give you access to our program.

<Steps>
    <Step title="Clone the project">
        ```
        git clone https://link
        ```
    </Step>
    <Step title="Go to the prokject directory">
        ```
        cd my-project
        ```
    </Step>
    <Step title="Install dependencies">
        ```
        npm install
        ```
    </Step>
    <Step title="Start the server">
        ```
        npm start
        ```
    </Step>
</Steps>
