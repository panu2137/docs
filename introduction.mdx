<Info>
    **Prerequisite** You should have installed Node.js (version 20.10.0 or
    higher).
</Info>

This quick setup will give you access to our program.

<Steps>
    <Step title="Clone the project">
        ```
        git clone https://github.com/granacik320/OpticsSimForMotorola.git
        ```
    </Step>
    <Step title="Go to the prokject directory">
        ```
        cd OpticsSimForMotorola
        ```
    </Step>
    <Step title="Install dependencies">
        ```
        npm install
        ```
    </Step>
    <Step title="Start the server">
        ```
        npm start
        ```
    </Step>
</Steps>
