The code displayed beneath the function name is the code we use to create the objects.
### Function: mirrorline
>
```java
elementsTypes['mirrorline'] = {
    create: (mouse) => {
        return {type: 'mirrorline', p1: mouse, p2: mouse}
    },
```

### Function: laser
>
```java
elementsTypes['laser'] = {
    create: (mouse) => {
        const p1 = mouse
        const p2 = objectUtils.point(mouse.x + 100, mouse.y)
        const dir = objectUtils.point(1, 0)
        const properties = ["position", "light"]

        return {type: 'laser', p1, p2, dir, properties, wavelength: 380, brightness: 1}
    },
```
