### Resize
>

| Property      | Description                           |
| ------------  |-------------------------------------- |
| width         | width of canvas                       |
| height        | height of canvas                      |


```javascript
    window.addEventListener('resize', () => {
            scene.resize(
                document.querySelector('.mainscene').clientWidth,
                document.querySelector('.mainscene').clientHeight
            );
    });
```
